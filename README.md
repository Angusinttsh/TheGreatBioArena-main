# TheGreatBioArena
哈囉大家好！歡迎參與午後倦容爐邊聚會的人來玩這款有趣的遊戲！

請大家務必在正式參與當天行程前完成自己的偉大生物！
# 遊玩方法：
首先想加入遊戲，你可以先複製一份已存在的生物當作您的基礎生物，並設計出您獨一無二的神奇物種！遊戲之初每種物種有100點基礎能量，召喚你的生物在隨機位置(總數為100/你生物的基礎總能量)，若無法整除則剩餘的能量會平均分配給你的生物。

## 你能做的調整有以下五種方向：
### 生物基礎物件：
您可以在元素物件區中瀏覽各個元素物件的功能，並透過組合他們為你的生物打好基礎。

注意了！元素物件的基礎能量愈高代表您使其成長的代價愈高，並非無腦的堆疊元素物件就能取勝！
### 顏色：
設定你的顏色，如果您不會使用RGB的話，請你離開！
### 移動邏輯：
如果你的基礎物件中帶有移動速度，代表你能夠有一套移動邏輯。
你須回傳兩個值：1、朝向(0\~360) 2、移動的速度(0\~1)
移動的速度會與你的速度數值相乘，當然愈快也會相應的消耗更多能量
### 繁殖邏輯：
你須回傳一個陣列，陣列的長度代表生物將一次分裂為幾個單位，而每個格子的值代表其在這些分裂的生物終戰的比值為何。
### energy_ratio
生物倍率，總能量/基礎總能量。其也會引響所有的數值倍率。
### note
這是你唯一可以儲存資料的地方，每個生物擁有獨立的字典，初始字典內包含一個遊戲預設的"_time"會是從出生到現在的秒數（精確度到0.1-0.3秒）。你可以自由新增、編輯字典內的參數（除了刪除"_time"），切記死亡後note將不會繼承。
## 元素物件
程式部分中有base_parts的部分，我們依序來介紹每個數值和物件的功能
### 數值
energy：基礎物件的能量，能量愈大代表你的升級和倍率體現會降低

volume：引響體積，可能可以使你更容易碰到周圍的生物

attack：可以對碰撞的生物造成傷害，務必記得，造成傷害本身會消耗額外能量

defense：相當於血量，數值歸零會直接死亡，會隨時間慢慢恢復

speed：移動的速度基礎值，移動會消耗額外能量

cost：每秒的能量消耗

photosynthesis_rate：葉綠素！

## 世界限制
### 葉綠素限制
葉綠素所能發揮的生產力會隨著生物總和能量的升高而逐漸降低。
比例為：

256:8

512:4

1024:2

2048:1

4096:0.5

8192:0.25

20000:0.1(約)

30000:0.05(約)
### 個體限制
單一個體的能量上限為250
能量下限為1 小於1 則自然死亡
### 物件限制
當生物元素物件達25件開始，會受到能量消耗暴增的制裁
